# Loom Product Roadmap

This document defines the initial roadmap for Loom.

---

# Version 0.1 — Foundation

Goal:
Establish project foundations.

Deliverables:
- project charter
- governance model
- roadmap
- issue templates
- PR templates
- branch strategy
- architecture definition
- repository setup

Status:
In Progress

---

# Version 0.2 — MVP

Goal:
Build the first usable CMS version.

Deliverables:
- authentication
- user management
- content CRUD
- page management
- initial admin dashboard
- SQLite development database

Status:
Planned

---

# Version 0.3 — Beta

Goal:
Prepare Loom for public beta testing.

Deliverables:
- media library
- role permissions
- audit logs
- CI/CD pipelines
- Azure deployment
- Azure SQL production database

Status:
Planned

---

# Version 1.0 — General Availability

Goal:
Production-ready public release.

Deliverables:
- stable API
- complete documentation
- security review
- performance validation
- public community onboarding

Status:
Planned
