# Loom Product Vision

This document defines the product vision for Loom.

---

# What is Loom?

Loom is an Open Source Headless CMS designed for modern teams building content-driven applications.

It is API-first, extensible, and cloud-ready.

---

# Product Purpose

Enable teams to manage and deliver content efficiently across multiple channels.

Loom should be:
- simple to start
- powerful to scale
- easy to extend

---

# Target Users

Primary users:
- developers
- content teams
- startups
- digital agencies

Secondary users:
- enterprises
- educators
- open source communities

---

# Value Proposition

Loom provides:

- modern headless architecture
- open source flexibility
- developer-friendly APIs
- easy deployment
- collaborative workflows

---

# Product Scope (MVP)

Initial scope:

- authentication
- user management
- content types
- content CRUD
- media library
- role permissions
- REST API
- admin dashboard

---

# Non-Goals

Loom is NOT:

- a website builder
- a no-code page builder
- a monolithic CMS
- a legacy plugin platform

---

# Positioning

"Loom is the Open Source Headless CMS built for learning and production."
