
# Loom Project Charter

## Overview
Loom is an Open Source Headless CMS built through collaborative learning.

The project is part of the Kramelworks Academy initiative, designed to provide real-world software development experience for developers and DevOps engineers.

---

## Mission
Build a production-grade CMS while training contributors through practical collaboration.

---

## Vision
Become a reference Open Source learning project where contributors gain real product development experience.

---

## Objectives
- Build a modern headless CMS
- Teach enterprise software development practices
- Teach DevOps and cloud deployment
- Create public portfolio opportunities
- Build a collaborative OSS community

---

## Target Audience
- Junior Developers
- Mid-level Developers
- DevOps Engineers

---

## Initial Stack
Backend:
- TypeScript
- NestJS

Frontend:
- Next.js

Infrastructure:
- Azure DevOps
- Azure App Service

Database:
- SQLite (dev)
- Azure SQL (prod)
